Trabalho de desenvolvimento de software

Requisitos estabelecidos

* Cadastro de funcionários

O cadastro deve conter todas as informaçôes necessárias dos funcionários

Após o cadastro o funcionário deve receber um login e uma senha com base em seu cargo

O sistema deve ser capaz de limitar o acesso do funcionário com base no seu cargo estabelecido no cadastro



* Recebimento e Classificação de produtos

Todos os produtos que chegar deve ser coletado (pelo coletor da empresa) e cadastrado no estoque pelo seu codigo de barras



* Gestão de estoque

Deve conter todos os produtos no estoque

suas respequitivas datas de validade

E sua quantidade



* Relatório financeiro

O relatório deve ser automatizado com base na entrada e saida de dados dos caixa

deve conter o total de vendas do mês

E os custos mais relevantes
